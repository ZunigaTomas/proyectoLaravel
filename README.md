## About Laravel

Una vez descargado la carpeta, la cual debe estar dentro de htdocs, realiza los siguientes pasos:

1 - Ejecutar "npm install". <br>
2 - Luego "composer install". <br>
3 - Crear la db. <br>
4 - Configurar .env. <br>
5 - Ejecutar "php artisan migrate --seed
